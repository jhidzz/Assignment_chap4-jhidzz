4장과제의 목표에 따라 recordFile을 이용하여서 1장에서 만든 3개의 객체를 불러와서 사용하였습니다. 
TestRecordFile.cpp을 사용하여 원래 Person에 적용 되었던 것을 제가 구현한 3개의 클래스에 맞게 고쳐서
테스트 하였습니다. 
그리고 3개의 각 클래스도 pack, unpack, clear, print 등 이번 chapter에서 필요한 버퍼에 대한 과정을 사용
하여서 원래 클래스에서 추가적으로 함수들을 구현하였습니다. 
그리고 recordfile을 템플릿으로 사용하여서 여러 클래스 타입에도 적용할 수 있다는 것을 알 수있습니다.

결과적으로 이번 과제에서는 전에 있던 클래스들을 활용하고, 버퍼사용에 맞게 함수를 추가하여 바로 파일 입
출력이 아닌 버퍼사용의 입출력을 구현 하였습니다.
